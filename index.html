Projects

Snake Game AI Experiment

GitHub Repo

This experiment implemented a simple version of the popular arcade game 'Snake', and tested various AI models on the game. The whole program was developed from scratch in Java. The models trained and tested on the Snake game were: a locally sensing agent, a gloabally sensing agent, a reinforcement learning agent, and a multilayer perceptron learning agent. You can view a report of the findings here.

TimeFlo (Pomodero Timer App)

GitHub Repo

A Pomodero style timer to assist with productivity and focus. Application has a GUI window and can run in the background while you work on tasks. Comes with customizable sounds and variable time settings for focus and break time. Developed with Python3 for Apple Macintosh computers.
